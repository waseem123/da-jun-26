Data Manipulation Language (DML) Statements - 

1. Data Insertion
	- INSERT : To add a new data to the database table
2. Data Updation / Change
	- UPDATE : To change an existing data from the database table
3. Data Deletion
	- DELETE : To delete existing data from the database table

Data while inserting and deleting - One or Many R ows
Data while updating - one or many values from one or many rows
